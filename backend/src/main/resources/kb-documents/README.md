# Knowledge Base Documents

Place your PDF, Markdown, or text files in the subdirectories below.
The IngestionService will auto-ingest all documents on application startup.

## Directory Structure

- `sops/` — Standard Operating Procedures (emergency triage, discharge, etc.)
- `insurance/` — Insurance plan details and coverage rules
- `triage/` — Symptom-to-urgency mapping and guidelines
- `departments/` — Department info (cardiology, general medicine, etc.)
- `facility/` — Visiting hours, maps, appointments, patient rights
